### Facts Learned
- Bash tool environment doesn't have `python`, `cat`, `tail`, `grep` — use full Python path and avoid piping to those commands
- Discord reply MCP tool uses `text` parameter, not `message` — wrong name causes "text.length" crash
- glTF format: `.gltf` (JSON descriptor) + `.bin` (binary keyframe/vertex data) — they're a pair
- Babylon Sandbox (sandbox.babylonjs.com) = best tool for previewing glTF animations with named clip dropdown
- Spine format = skeletal animation for 2D; glTF = skeletal animation for 3D — different ecosystems
- Sketchfab "Take01" single-timeline exports = annoying, require manual frame mapping
- Mixamo works for humanoids but not four-legged creatures/dragons
