# Demo POS 2.2 New Features

With the release of Demo POS 2.2 on 02/18/2025, there are new tools available to assist in troubleshooting.


 

## POS Items Edit Tools:


We now have more options in the Edit POS Items section (Tools>POS Items):


            **Add **– Adds new line to enter Item Data


            **Copy **– Copies highlighted item to a new row


            **Delete** – Deletes currently highlighted row


 **Export **– Exports entire list to a .csv


 **Import **– Import items from compatible .csv


Simulate:


 Simulate (Tools>Simulate) allows Demo POS to simulate service utilizing the currently loaded POS Items list.


*Note: At this time, it will only send items as Parent items. There are no Parent/Child relationships


 


 


- Delay between POS messages – The speed at which items are sent from the POS

- Courses per order – Minimum and Maximum number of courses are sent per order

- Items per order – Minimum and Maximum number of items per order

- Item Selection – Here you have several options for how the simulator chooses items to send

- Choose items randomly from all tabs – This option will randomly choose items from all tabs regardless of type

- Choose items from these tabs – This option allows you to choose specific tabs to pull items from

- Choose from these Item ID’s – You may specify specific item ids from the POS Items that you want the simulator to choose from.

- Preserve order of IDs - When this setting is enabled it will send orders in order of the comma separated items listed in the above setting


- This section is where you can see the individual items per order as it is being sent to Kitchen
