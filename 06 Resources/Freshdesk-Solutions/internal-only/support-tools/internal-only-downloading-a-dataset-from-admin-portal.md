# |Internal Only| - Downloading a Dataset from Admin Portal

**Sites on CSK 2020.12+ will automatically upload the live Data Directory of any site to the Admin Portal. **


This allows us do download them when needed.


**To download the dataset:** 


Log into the [Admin Portal]


Navigate to the site:


Expand the Site Settings on the left side:


Click on Kitchen Storage Management:


Click on Download Dataset:


**NOTE:**


If you receive an error screen then double check the installed version as it typically means that they are not on a high enough version to support this function.
