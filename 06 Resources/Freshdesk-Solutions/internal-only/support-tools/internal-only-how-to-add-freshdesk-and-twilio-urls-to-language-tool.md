# *|Internal Only| - How to Add Freshdesk and Twilio URLs to Language Tool

In order for the Language tool add on to be able to review messages typed in tickets and chats with customers you will need to add the Freshdesk and Twilio Flex URLs to the site access menu in the Language Tool details in Chromes Extension manager.


This can be found via the three dots just under the X button in top right corner of Chrome and selecting "Extensions" and then "Manage Extensions"


On the next screen, select the details option under Language Tool:


Then in the menu for Site Access hit the Add a New Page option and add **https://flex.twilio.com/* **and **https://qsrautomations.freshdesk.com/*** ** **(Including the *)


If successful, you should see a blue checkmark in a blank chat box in Twilio Flex or Freshdesk when you have a text entry field selected.
