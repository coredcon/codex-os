# *|Validation|License Exceptions

**License Exceptions**


**All licenses must be synced from the Portal, both Perpetual and SaaS.  If there is a site that is not in the Portal, please email to either ClientServices or PartnerAssist to have the site added to the Portal.  If there is an emergency license need and the customer is not in the Portal, approval to send a license must come from CJP or KJK only.**


**A&W - **Licenses should be created Auto-Renew (SaaS) even if showing Asset codes in Salesforce.  SoS is SaaS, so the entire license for A&W locations should be Auto-Renew.


**BJs Restaurants - **If product code 001-7956 is present in subscriptions, please be sure to enable TeamAssist Integration on the license.


**Boston Pizza** – Now SaaS and installing CSK 2022 and CSH 6, 2022 (if the site has CSH product codes as not all Boston Pizzas use CSH).  DineTime POS Interface should also be added to the license per KJK IF the site is licensed for CSH.


**Chimney Rock Inn** - Always enable TeamAssist Integration (approved by KJK).


**Moxie’s \ Chop** – Always gets TeamAssist Integration, even if not showing in LMTS2.


**PF Changs** - TeamAssist Legacy ConnectSmart Validator location on PF Changs image is here: C:\QSR Install Files\Team Assist\EN-US\ConnectSmartValidator or this path C:\Source\TeamAssist\TeamAssist Installer\ConnectSmartValidator.


**Red Robin International Inc (Corporate Stores)** - (01/06/25) - Per Tony Griffin, If any Red Robin International Inc location has the new CS Host product code, 001-5254, or any of the Legacy DineTime Enterprise Services product codes listed in the table below, Validation-only support will be provided through 12/31/2025.  This includes CSK perpetual license validations for supported versions (currently CSK 2019 and higher).  Reminder, when creating a perpetual CSK license for Red Robin International Inc, do not put the station count in non-QSR unless a non-QSR hardware product code is present in SF.


****The Red Robin franchisees below are eligible for full Support (technical and validation)****


- Restaurant Development Company of Medford LLC (Per CJP via email on 09/15/23)


- Infinity RRGB Ventures Inc (Per CJP via Slack message in support_team channel on 12/22/23)


**SeaWorld of Texas (SID 16569) - **(02/26/24 per NW) Site may be licensed for CSK 2020 and CP 2020, despite what is in SF.  The Corporate license allowed for CSK 2020 and this is the version they were running at the time of conversion from Corporate to Portal licensing. Farris Saadah and Connor Joffe are working to get a new PA signed.
