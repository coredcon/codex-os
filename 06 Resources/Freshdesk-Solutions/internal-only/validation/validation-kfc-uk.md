# *|Validation| KFC_UK

**KFC_UK** - (04/16/25) - All KFC_UK sites have been updated to **001-7955** and **30 stations**.  **Curbit** has also been added to each KFC_UK site. This will need to be validated in Salesforce and **Events Web Service** will need to be enabled in the license. All KFC_UK licenses are currently up-to-date with this information.  We cleaned out over 400 old licenses out of LMTS before we updated everything.  If they email, or call, for a new license, and it wasn't a server swap that prompted the new license, please make sure you send the new canned response for KFC_UK Validation Request before you issue a new license.  Most likely, they have a NIC issue that caused the site code to change.  


Please also add **CSK 2019/2022/2025** and **CP 4/2022/2025** to all licenses
