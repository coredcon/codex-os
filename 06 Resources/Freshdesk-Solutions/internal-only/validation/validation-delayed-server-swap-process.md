# *|Validation| Delayed Server Swap Process

**OSI/Bloomin' Brands** - _(per NW 02/26/24)_ OSI/Bloomin' Brands will be rolling out new servers to many locations starting late March/early April 2024.  There will be some ones and twos prior to then as they ensure their process is air tight.  Please use the following process for these requests: Leave the existing license active, create the new license as usual (SaaS) and update SF with the new site code.  When you reply letting them know the new license has been created and is ready for sync from the Portal, please request they send a reply once the swap is completed so the old license may be deleted from LMTS Portal and the ticket updated and closed.
