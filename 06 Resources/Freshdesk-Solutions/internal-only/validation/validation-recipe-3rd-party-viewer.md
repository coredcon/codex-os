# *|Validation| Recipe 3rd Party Viewer

I wanted to share this with you, as it came up again this week. What is stated below is what is approved by Tony.


 


- Recipe 3rd Party Viewer              

- This will continue to be free and apart of CSK (like we have been doing)

- As long as you are on the latest product codes for base, pro, or premium CSK bundle


- No separate product code to track this service at $0.00

- This is the product description being referenced


- Team Assist 1, Team Assist 2, CS Recipes:  Should not be used anymore and CS Recipes Plus should only be sold


The only caveat is an existing customer that is using one of the above and is opening a new site


If a customer wants to use this, they must be on a supported software version and have the latest product codes for Base, Pro or Premium (001-7954, 001-7955 or 001-7956 respectively).  


 


The customer may email Validation directly or send a request to their Account or Channel Manager and they will send it on to Validation.  If you receive a request to enable this feature (and the aforementioned criteria is met), please toggle on the “TeamAssist Integration” option on the Kitchen license in LMTS Portal.
