# *|POS| Display VideoMX Setting in Aloha Configuration

Below is the location to enable the setting that allows the user to view the VideoMX enable/disable options.


This is located Maintenance > Business > Corporate Administration > Select Display VideoMX


NOTE: This setting may have been moved in newer versions of Aloha
