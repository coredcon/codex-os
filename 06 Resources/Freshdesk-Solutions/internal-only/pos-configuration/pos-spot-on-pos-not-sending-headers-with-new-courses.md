# *|POS| - SPOT ON POS not sending headers with new courses

If you have a site using SPOT ON POS that is not sending header messages when they send a new course there is a setting in the SPOT ON **Print Routing Options** for **Auto Apply Course on Send.**


Have them enable that setting and verify that that the headers are sending correctly once that is enabled.
