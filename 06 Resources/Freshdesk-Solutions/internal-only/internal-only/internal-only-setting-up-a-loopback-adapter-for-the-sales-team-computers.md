# *|Internal Only| - Setting Up A LoopBack Adapter For The Sales Team Computers

This may also need to be done if the adapter disappears for any reason.


 It seems to do this randomly and on everyone’s machines.


All of the apps on the computer should be bound to either 192.168.1.100 or possibly 192.168.10.100


To set this up:


Search hdwwiz from the start menu and run the program.


Click Next.


Click "Install hardware manually from a list"


Select Network Adapters.


Under the manufacturers choose Microsoft.


Choose Microsoft KM-Loopback Adapter.


Click Next.


Click Next.


Restart the services.
