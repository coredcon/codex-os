# Data Needed from Old Server to New Server

If using the same dataset and hardware configuration copy these folders from the old server and add to the new server in the same locations.  


 


**Folders needed for ConnectSmart Kitchen 8 through 2024 **


C:\ProgramData\QSR Automations\ConnectSmart\KitchenServer\Data


C:\ProgramData\QSR Automations\ConnectSmart\KitchenServer\Datasets


**Recommended:**


C:\ProgramData\QSR Automations\ConnectSmart\KitchenServer\BinManager


C:\ProgramData\QSR Automations\ConnectSmart\KitchenServer\Databases


Folders needed for ControlPoint: 


C:\ProgramData\QSR Automations\ConnectSmart\ControlPointClient\Data 


C:\ProgramData\QSR Automations\ConnectSmart\ControlPointServer\Data 


C:\ProgramData\QSR Automations\ConnectSmart\ControlPointServer\Templates 


**Step 1:** Go to File Explorer.  


⊞ Window+E key opens File Explorer


Or Search for File Explorer


**Step 2: **Create a compressed file in File Explorer


**Step 3: **Under this PC go to desktop (this may be in Quick Access or go to step 6 and Select Desktop)


**Step 4: **Right click in the blank part of the Desktop window > Select New > Select Compressed Folder (zip)*


 *Please create Folder with the restaurant name, store # if applicable and date


**Step 5:**  Open a new File Explorer window. 


⊞ Window+E key opens File Explorer


Or Search for File Explorer


**Step 6:** Go to** This PC ** or the alternative is to copy paste the path into the drop down menu 


**Step 7:** Select **Window’s (C:)** drive


**Step 8:**  Select **ProgramData**


**Step 9:** Select **QSR Automations**


**Step 10:** Select **ConnectSmart**


**Step 11:** Select **KitchenServer**


**Step 12:** Drag folder over to compressed folder we created earlier under a separate window.


                * Please include: **Data, Databases, Datasets,** and **Bin Manager**


**Step 13:** Click on the compressed folder to confirm if the folders copied.


**Step 14:** Create compressed folder for ConnectSmart, ControlPointClient, and ConnectSmart ControlPoint Server


**Step 14:** Once compressed folders have the files, send over to our team via your ticket or email [support@qsrautomations.com]
