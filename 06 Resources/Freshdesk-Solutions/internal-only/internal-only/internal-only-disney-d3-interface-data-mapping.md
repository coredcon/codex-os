# *|Internal Only| - Disney D3 Interface - Data Mapping

What is it?: 


- A few years ago, Disney contracted our Dev team to write an interface for them that pushes DineTime/ ConnectSmart Host Visit Data to their data warehouse.  The contract for this integration was completed and the source code was delivered to Disney, so we do not technically offer support for this integration (we cannot add bug fixes or enhancements without a new Dev contract).  

However, Disney occasionally reaches out with questions about the interface, so the Spec docs for data mapping are attached below.


** Disney has since re-written the D3 interface and it goes by a new name now. **
