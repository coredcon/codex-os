# Determining Versions of QSR Software Installed:

Easiest way to determine what version of QSR Software site has installed for licensing.  


To determine the** ControlPoint **version:  


Navigate to C:\Program Files (x86)\QSR Automations\ControlPoint\ControlPointServer\Bin 


Right click on ControlPointServer.exe 


Select properties  


Click on Details tab 


This tab should list the File version 


 


To determine the **ConnectSmartKitchen **version:  


 


Navigate to C:\Program Files (x86)\Qsr Automations\ConnectSmart\KitchenServer\Bin   


Right click on KitchenServer.exe 


Select properties 


Click on Details tab 


This tab should list the File version 


 


To determine the** ConnectSmartServer (Host) **version:** ** 


 


Navigate to C:\Program Files (x86)\QSR Automations\ConnectSmart\HostessServer 


Right click on ConnectSmartServer.exe  


Select properties  


Click on Details tab 


This tab should list the File version
