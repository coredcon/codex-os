# *|Internal Only| - Determine which .NET Framework versions are installed

To determine exactly what version of >NET is installed on a computer, follow these steps:


Search Features in the start menu and select Turn Windows features on or off


 


This can also be found here: Control Panel\All Control Panel Items\Programs and Features and then select Turn Windows features on or off  on the left


The installed versions of .NET will be visible in that form:
