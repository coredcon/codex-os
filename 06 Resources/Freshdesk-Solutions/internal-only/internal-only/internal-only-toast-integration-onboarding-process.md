# *[Internal Only] - Toast integration onboarding process

This is the process for customer's to kick off, enabling the Toast integration.


- To enable the Toast integration with QSR, the customer should add the integration from their Toast portal. 

- Toast provided the following link for instructions **https://central.toasttab.com/s/article/Adding-or-Removing-an-Integration-with-Toast-Partner-Connect  **


- Once the integration has been added by the customer, QSR receives an automated email that contains the Site name, location, and GUID. 

- It also includes the contact information of the user who added the integration. 


- Once we receive that email, the integration is then enabled by Data Management within 1-2 business days. 

- You can either check admin to see if the integration has been enabled if the customer claims that they did their part.  You can also ask Tech projects to verify if we've received the activation email.
