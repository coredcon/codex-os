# *|Internal Only| - FreshDesk CSK Ticket Category Rundown

The following is a list of definitions for each CSK Ticket Category in FreshDesk that Kathi has provided. Please use this list as a reference for all CSK tickets to ensure that you are selecting the proper category when working on CSK issues.


- **Configuration** - Questions regarding any settings / options in the CSK Builder to manage the displays and operational behaviors of the solution.


- **Routing** - Specific questions regarding how items are configured to send to specific stations or views at the site.   Example: The customer has added new menu items and needs assistance ensuring these route / display on the proper station in the kitchen.


- **Server Replacement** - Questions regarding moving the previous data files used by CSK to the new Server; Compatibility questions in the event the new server has a different Windows Operating System; network configuration within our applications, etc.


- **Installation** – Assistance with a new installation whether it be a new site or a server replacement.  Selecting this issue indicates QSR Support completed the entire installation process with the Customer online for training.


- **Troubleshooting** – Assistance diagnosing site issues in general not related to configuration or routing.  Selecting this issue indicates the customer has implemented a feature within our solution that is not behaving as expected.  It could, and often time does, include helping them troubleshoot connectivity issues within their network, or POS issues, that are not QSR problems per se.  (e.g. Stations not connecting to the QSR Services due to firewalls or anti-virus software, POS terminals failing to send items / orders to QSR Services, and or POS terminals sending items incorrectly).


- **Upgrade** – Questions regarding moving to a newer version of CSK.  (e.g. Compatibility of existing configuration files with the newer application version, questions about new functionality available with the upgrade, etc.).


- **TLS1.2** – This is for use in reference to the EoL for CSK 8 and older in addition to any communication related to the TLS 1.2 communication that was sent out. This can apply to any version of CSK. Even if their version of CSK isn’t EoL’d they could still have questions about enabling the correct version of TLS etc.
