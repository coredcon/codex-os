# |Internal Only| Exporting a Capture to CSV

An underused feature of Capture Player is the Export CSV function. This allows you to create a spreadsheet of all events in the Capture file to aid in troubleshooting.


 


After choosing this option, you can choose the relevant columns you would like to include. 


 


After choosing this option, you can choose the relevant columns you would like to include. Then choose the File Path you would like to save this to and click OK


 


My recommended Colums:


 


Subfunction Value Description list is available in a Solutions Article.


[https://qsrautomations.freshdesk.com/a/solutions/articles/17000059621]


Helpful values are:


36 - CancelTransaction


37 – CancelItem


40 – TotalTransaction


41 – TenderTransaction


 


When this spreadsheet is helpful


- When you know limited information about the transaction (ieTable/Customer Name/approximate time)

- You want to search multiple Transactions spread out throughout the Capture

- Providing concise information back to a customer without having to use screenshots from Capture


**NOTE: You will only get all data on a QSRSock Capture. It will not pull items from a cloud/API order.**
