# *ConnectSmart Kitchen Server 2019.1.5 (Released 4/4/2019)

**Bug Fixes:**


Fixes possible thread deadlock causing reported Kitchen Server freeze


[https://qsrautomations.freshdesk.com/a/tickets/161561] 


**New Features/Enhancements:**


Supports new Bin Metric for dashboards "Used All Day"


- Added support in KitchenServer for new bin metric, called "Used All Day." This metric requires the selection of a Bin ID, and then keeps a running tally of all items used for that Bin Item since the last EOD. The metric will only reset during End of Day processing.
