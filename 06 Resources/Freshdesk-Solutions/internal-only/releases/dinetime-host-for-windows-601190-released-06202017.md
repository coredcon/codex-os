# *DineTime Host for Windows 6.0.119.0 (Released 06/20/2017)

**New Features/Enhancements** 

- Drops support for Windows XP and Windows Server 2003 **TT#: 28744**

- Syncs DineTime site license from Enterprise **Jira#: GM-106**

- Supports page a manager via SMS **Jira#: GM-52, GM-55, GM-118 GM-47 Request: BJ’s **_Note: Feature is also supported in Host (iPad) 4.0+, Assist 2.0+_

- Maintains selected Reservation Time slot on Edit Reservation when Party Size changes **Jira#: GM-50 Request: Martini Italian Bistro **_Note: Feature is also supported in Host (iPad) 4.0+_

- Adds External Alerts XML interface **Jira#: GM-105 Request: Darden**

- Supports In-client Report UI **Jira#: GM-34,  GM-114, GM-48, GM-46, GM-111, GM-112 Request: Darden **_Note: Feature is also supported in Host (iPad) 4.0+_

- Enables guest record edit from reservations for past days in Reservation Book **Jira#: GM-31 Request: CST/Dishoom (001-00-054994)**

- Supports a user configured SQL memory cap **Jira#: GM-110 Request: Cheesecake Factory**

- Adds Walk Away buttons to “Priority Is Wait Listing” greeter **Jira#: GM-43 Request: Cheesecake Factory**

- Supports GuestView 6.0 **Jira#: GM-147, GM-142, GM-143-46, GM-140, GM-141, GM-149**

- Sends DeviceID associated with table to PosiToluchTSConnect **Jira#: GM-154 Request: Boston Pizza**

- Supports OpenTable VisitSource **Jira#: GM-32**

- Adds support for Service Discovery **Jira#: GM-107, GM-309**

- Updates the Visual C++ versions included with the full installation **Jira#: GM-305, GM-307**

- Improves error handling during EOD processing **Jira#: GM-124 Reported: Disney (108646)**
** ****QSRCommon Tools** 

- Supports a custom installation path for QsrCommonTools MSI **Jira#: ECS-322**

- Adds new licensing product code for DineTime 6.0 in QsrLicenseManager.exe **Jira#: ECS-325**

- Supports the new service discovery service in DineTimeRegistration.exe **Jira#: ECS-321**

- Adds ability to specify Primary/Secondary Kitchen Server registration in DineTimeRegisration.exe **Jira#: ECS-319**

- Adds new licensing product code for Kitchen Server 7.0 in QsrLicenseManager.exe **Jira#: ECS-326**

- Updates DineTimeRegistration.exe to support Service Discovery **Jira#: ECS-364**
