# *ConnectSmart Kitchen Server 6.4.110  (Released 03/31/2017)

Release - ConnectSmart Kitchen Server 6.4.110


**New Features/Enhancements**** **

- Adds support for TeamAssist 2.0 **Jira#: CSK-118, CSK-119, QCT-7 **_Note: Requires TeamAssist 2.0+_

- Adds ability to specify bin period or hold time as look ahead time for projections** Jira#: CSK-128, CSK-129 Request: Boston Pizza**

- Allows spy views to accept keypad/dashboard input and forward input to the master view** Jira#: CSK-117, CSK-120, CSK-123, CSK-127 Request: Boston Pizza**

- Adds setting to allow the item background color to override the cursor background color** Jira#: CSK-156, CSK-157 **_Note: Enterprise configuration available in Portal 5.10+_

- Adds Guest Count to the list of available fields in Receipt Templates **Jira#: CSK-153, CSK-166 **_Note: Enterprise configuration available in Portal 5.10+_

- Allows CSK alternate path installation** Jira#: CSK-151 Request: Brinker**

- Updates End User License Agreement in ConnectSmart installers** Jira#: CSK-184**
** ****Bug Fixes** 

- Fixes issue where reprioritized items were not always sorting to top ** Jira#: CSK-170 Reported: CST (104057)**
