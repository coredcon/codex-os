# *ConnectSmart Kitchen Server 7.3 (Released 03/27/2018)

Release - ConnectSmart Kitchen Server 7.3

Server
** ****New Features/Enhancements**** **

- Adds option to allow parent status to be based on its component item’s status including statuses other than prep **Jira#: **[**KS-975**], [**KS-1149**] **Request: Darden (**[**KS-991**] **/ **[**DE-670**]** / **[**122560**]**)**

- Add option to disable delaying future orders **Jira#: **[**KS-985**] **Request: Darden (**[**KS-988**]**)**

- Supports setting to transition View Metrics between Activity Levels **Jira#: **[**KS-1137**] **Request: Darden (**[**KS-1130**]**)**

- Supports Routing Category Display Exceptions on receipts **Jira#: **[**KS-1110**] **Request: Cracker Barrel (**[**KS-1109**] **/ **[**DE-541**]** / **[**117674**]**)**
** ****Bug Fixes**** **

- Fixes Kitchen Server talking a long time to load large datasets **Jira#: **[**KS-984**] **Reported: Darden (**[**DE-689**]** / **[**122913**]**)**

- Fixes Kitchen Server crash during shutdown related to HTTP **Jira#: **[**KS-1148**] **Reported: Mod Market (**[**DE-831**]** / **[**127724**]**) **_Note: This escalation was already closed with the release of this fix in CSK 7.2 R2 on 2/27._
