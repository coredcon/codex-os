# *DineTime Host for Windows 5.6.107.0  (Released 10/14/2016)

**New Features/Enhancements** 

- Support for combined Guest Notes, Visit Notes, Food Allergies tab **TT#: 226771**  **Request: Disney** _Note: Requires Host (iPad) 3.8, Assist 1.7_

- Utilizes new settings and calculations related to GPH quoting in Server Sites **Request: Darden** _Note: Requires Host (iPad) 3.8_

- Include unarrived parties within X minutes **TT#: 26691**

- Color indicator for parties within X minutes **TT#: 27619**

- Match waiting parties to tables before providing quote **TT#: 26694**

- Inclusion of dirty/closed tables in available tables **TT#: 26692**

- Discount unarrived party slots based on configured no-show % **TT#: 26693**

- Estimate arrival time based on settings in Operating Hours/Periods **TT#: 26804**


- Supports fully customizable SMS Notifications in Server sites **TT#: 27020, 27021** **Request: Cracker Barrel** _Note: Requires Host (iPad) 3.8, Assist 1.7_

- Only cleans dirty tables during EOD instead of all open tables **TT#: 28001**  **Request: Disney**
** ****Bug Fixes** 

- Fixes possible crash when a Floor Plan exists without any Room Views defined **TT#: 28395**  **Reported: Cracker Barrel (001-00-066915)**
