# *DineTime Host for Windows 2019.1.5 (Released 06/28/2019)

**New Features/Enhancements** 

- Updates versioning to 2019.x **Jira#: **[**GM-2176**] **(**[**GM-2175**]**)**

- Connected Clients improvements **Jira#: **[**GM-2034**], [**GM-2035**]

- Improves security of in-store personal data **Jira#: **[**GM-2417**], [**GM-2433**] **(**[**GM-1656**]**)**

- Purges Historical Quoting data at End of Day and reduce retention of data from 15 months to 24 weeks **Jira#: **[**GM-2425**], [**GM-2546**] **(**[**GM-1656**]**)**

- Limits Guest Search to valid data **Jira#: **[**GM-2295**], [**GM-2303**], [**GM-2311**] **(**[**GM-1679**]**)**

- Updates installation to 2019.1 version of QSR Common Tools **Jira#: **[**GM-2570**] **(**[**GM-2175**]**)**
