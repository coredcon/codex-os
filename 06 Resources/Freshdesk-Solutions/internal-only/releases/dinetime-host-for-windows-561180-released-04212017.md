# *DineTime Host for Windows 5.6.118.0 (Released 04/21/2017)

**New Features/Enhancements** 

- Supports OpenTable visit source **Jira#: DTW-69**

- Only shows 15 minute increments on reservation time override for OpenTable visit source **Jira#: DTW-90**

- DineTimePOSInterface sends DeviceID associated with Table to PosiTouchTSConnect **Jira#: DTW-92 Request: Boston Pizza**
** ****Bug Fixes** 

- Fixes issue where Aloha client would stop working after losing connection to DineTime Server **Jira#: DTW-101 Reported: Buffalo Wild Wings (108079)**

- Fixes issue with Historical Sample processing around daylight time transition **Jira#: DTW-102 Reported: Red Lobster (108388)**

- Improves error handling during EOD processing **Jira#: DTW-111 Reported: Disney (108646)**

- Fixes issue where guest update API Call was failing to update Loyalty information **Jira#: DTW-117 Reported: BJ’s (108772)**
