# *DineTime Host for Windows 5.4.125.0  (Released 03/14/2016)

**New Features/Enhancements** 

- Changes server list to use natural sort **Request: BJ’s**

- Allows bulk table edit in DineTime Host **Request: BJ’s**

- Adds auto-login functionality **Request: BJ’s**

- Changes table status for moved tables to the status of the section to which it is being moved **Request: BJ’s**

- Utilizes pre-configured text messages for manager paging **Request: BJ’s**

- Phone number searches now wait for a full valid phone number instead of searching after 3 digits are entered

- Supports ability to Quick Seat an anonymous party at a table in TableBoss **Request: CST**
 **Bug Fixes**** **

- Fixes issue where notification type was being set to verbal instead of SMS **Reported: BJ’s (001-00-054697)**

- Fixes issue where Visit Quick Notes using the same image caused a sync error **Reported: Dishoom (001-00-055175)**

- No longer enforces email validation on read-only guest fields in greeter **Reported: BJ’s**
**(001-00-056043)**

- Adds Quote String to SMS messages originating from BackOfficeCallAheadInterface **Reported: BJ’s (001-00-056072)**

- Fixes issue where BackOfficeCallAheadInterface was not properly dealing with call ahead times set as UTC **Reported: BJ’s (001-00-056387) **_Note: Fix addresses issue as described by BJ’s but it has not been verified by QA_
