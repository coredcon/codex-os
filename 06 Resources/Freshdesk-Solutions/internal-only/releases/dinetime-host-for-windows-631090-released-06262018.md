# *DineTime Host for Windows 6.3.109.0 (Released 06/26/2018)

**New Features/Enhancements** 

- Defaults State/Province dropdown in Guest Book to that of the site  **Jira#: **[**GM-518**]** Request: Joey’s (**[**GM-517**] **/ **[**DE-466**]** / **[**114177 **]**)**

- Adds additional logging for diagnostic purposes **Jira#: **[**GM-1424**]**, **[**GM-1425**]_ Note: Privately released to Darden in 5.6.124.0 on 3/14._

- New Pricing Guide Product Code 001-5027: License bulk Team Member Imports via XML **Jira#: **[**GM-662**] **(**[**ECS-619**]**)**

- New Pricing Guide Product Code 001-5025: License Premise Pager capabilities **Jira#: **[**GM-700**] **(**[**ECS-619**]**)**

- New Pricing Guide Product Code 001-5026: License use of DineTimePOSInterface **Jira#: **[**GM-1359**] **(**[**ECS-619**]**)**

- Supports 64-bit integers for Transaction Number and converts to 32-bit in Polaris **Jira#: **[**KS-1494**]** Request: CST/Big Mamma/Ikentoo POS (**[**GM-1518**]** / **[**DE-977**]**)**

- Hides Reservation Book, Reserved Tab, and Add Reservation when subscription level doesn't support it **Jira#: **[**GM-846**], [**GM-1399**]
** ****Bug Fixes** 

- Fixes crash related to background sorting of guest list  **Jira#: **[**GM-1423**]** Reported: Darden (**[**DE-847**]** / **[**129397**]**) **_Note: Privately released to Darden in 5.6.124.0 on 3/14._

- Updates QsrLicenseInterop.dll to fix licensing issue with certain date formats **Jira#: **[**GM-1434**]** Reported: Boston Pizza (**[**DE-720**]** / **[**124173**]**)**

- Removes mouse input capture from keyboard, making it more consistent with Windows on-screen keyboard** Jira#: **[**GM-1398**]** Reported: Darden (**[**DE-724**]** / **[**124430**]**) **_Note: Privately released to Darden in 5.6.124.0 on 3/14._

- Pins Unhandled Exception message to top of client** Jira#: **[**GM-1426**]** Reported: Darden (**[**DE-724**]** / **[**124430**]**) **_Note: Privately released to Darden in 5.6.124.0 on 3/14._

- Adds a global MouseUp event handler with the purpose of releasing any lingering mouse capture.  **Jira#: **[**GM-1469**] **Reported: Darden (**[**DE-724**]** / **[**124430**]**) **_Note: Privately released to Darden in 5.6.126.0 on 3/30._

- Fixes no longer being able to select the Party Type as "Call Ahead" **Jira#: **[**GM-1468**]** Reported: Darden (**[**DE-929**]** / **[**131007**]**) **_Note: Privately released to Darden in 5.6.128.0 on 4/6._

- Shows the print button as soon as the calendar date rolls over to the same date as the reservation **Jira#: **[**GM-1487**]** Reported: Disney (**[**DE-976**]** / **[**133651**]**)**

- Removed all logic in ZoomPan Control that manipulates the IsHitTestVisible property in attempt to resolve FloorPlan lockup.  **Jira#: **[**GM-1484**]** Reported: Darden (**[**DE-308**]** / **[**111491**]**) **_Note: Privately released to Darden in 5.6.129.0 on 5/2_

- Fixes TableScout messages failing to process  **Jira#: **[**GM-1541**]** Reported: LuLu’s Landing (**[**DE-1074**]** / **[**136711**]**)**
