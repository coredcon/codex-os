# *DineTime Host for Windows 5.6.112.0  (Released 11/18/2016)

**New Features/Enhancements** 

- Sets AlohaClient service start up type to Automatic during installation instead of Manual **TT#: 28495**

- Adds additional logging for when Shifts or Floor Plans are deleted** TT# 28595**
** ****Bug Fixes** 

- Fixes issue where DineTime Aloha Interface wouldn't start up for first time if there were any existing POS files in the \Aloha\TMP dir. **TT#: 27179**  **Reported: Red Robin, Doolittle’s (001-00-060544)**

- Fixes error related to editing reservations and time zone **TT#: 28818**  **Reported: CST/Jamie’s Italian (001-00-068952)**
