# *DineTime Host for Windows 6.4.105.0 (Released 09/28/2018)

**New Features/Enhancements** 

- Adds “Guest Request” to Visits DTO **Jira#: **[**GM-600**] **(**[**MAS-1396**]**)**

- Supports adding client software version to logging **Jira#: **[**GM-998**] **(**[**GM-997**]**)**

- Supports logging generated quote values in HostessServer.log upon Enterprise sync **Jira#: **[**GM-1442**]

- Displays visit origin in DineTime **Jira#: **[**GM-1427**], [**GM-1429**], [**GM-1604**], [**GM-1684**], [**GM-1685**] , [**GM-1686**] , [**GM-1696**]  **Request: BJ’s, Doolittle’s, Winston Grille (**[**GM-1414**]** / **[**DE-659**]** / **[**DE-583**]** / **[**119682**] **/ **[**DE-482**]** / **[**113727**]**)**

- Logs generated quote values in HostessServer.log when they are synced to Enterprise **Jira#: **[**GM-1442**]

- Displays both Pager ID and Actual Wait time on Seater Receipt  **Jira#: **[**GM-1534**] **Request: Darden (**[**GM-1533**]** / **[**DE-858**]** / **[**128537**]**)**

- Fully supports 64 bit integers instead of converting them to 32 bit  **Jira#: **[**GM-1500**] **Request: Ikentoo POS/Big Mamma (**[**DE-977**]**)**

- Adds support for new custom guest tab sorting method "Estimated Seat Time"  **Jira#: **[**GM-1581**], [**GM-1582**] **Request: Market Place Grill (**[**GM-1578**]** / **[**DE-856**]** / **[**128509**]**)**

- Adds the Open Menu Count metric as one of the possible Dashboard Items **Jira#: **[**GM-658**] **Request: Joey’s (**[**GM-1573**]** / **[**DE-663**]** / **[**122269 **]**)**

- Displays takeout tab when using Custom tabs if the takeout tab setting is enabled  **Jira#: **[**GM-1651**]
** ****Bug Fixes** 

- Refreshes data in the Servers tab on both polling and section update **Jira#: **[**MAS-1276**] **Reported: Red Lobster (**[**DE-1005**]** / **[**134798**]**)**
