# *DineTime Host for Windows 5.6.122.0 (Released 08/15/2017)

**Bug Fixes** 

- Fixes issue where FloorViewer could get into a state where all mouse input is disabled  **Jira#: GM-324 Reported: Darden (111491) **_Note: A private release containing this fix, 5.6.119.0, was provided to Darden on 06/16/2017_

- Changes read/write lock to allow concurrent reads to address performance issues  **Jira#: GM-378 Reported: Cactus Club (113369) **_Note: A private release containing this fix, 5.6.120.0, was provided to Cactus Club on 07/20/2017_

- Improves the performance of reservation searches  **Jira#: GM-379, GM-399 Reported: Cactus Club **_Note: A private release containing this fix, 5.6.122.0, was provided to Cactus Club on 08/01/2017_
