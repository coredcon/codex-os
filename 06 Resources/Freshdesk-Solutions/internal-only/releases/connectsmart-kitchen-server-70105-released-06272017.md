# *ConnectSmart Kitchen Server 7.0.105 (Released 06/27/2017)

Release - ConnectSmart Kitchen Server 7.0.105


**New Features/Enhancements**** **

- Supports Cloud POS **Jira#: KS-123, 124, 130, 131, 146, 146, 147, 149, 152, 154, 162, 164, 166, 169, 234**

- Supports Cloud Kitchen **Jira#: KS-136, 137, 142, 529, 570, 586**

- Adds sorting mode options to support dual lane drive-thrus **Jira#: KS-165 Request: KFC Australia **_Note: Enterprise configuration is supported in Portal 5.11.112.0+_

- Adds support for Service Discovery **Jira#: KS-157**

- Adds support for  Future Order "pickup time" or "desired completion time" field(s) **Jira#: KS-126 **_Note: Future order functionality is not yet supported in CSK_

- Adds access to the Events Web Service to allow receiving an ItemBumped event **Jira#: KS-572 Request: Sweetgreen**

- Adds setting to enable/disable guest full name as Customer Name on display **Jira#: KS-66 **_Note: Enterprise configuration is supported in Portal release TDB_
** ****Bug Fixes** 

- Fixes possible transaction deletions when duplicate "NewHeader" received from POS  **Jira#: KS-234 Reported: Greenleaf Radisson (107415) **
