# *DineTime Host for Windows 7.1.105.0 (Released 04/02/2019)

**New Features/Enhancements** 

- Adds support for new data type "Time Left In Quote" on Guest Tabs **Jira#: **[**GM-1548**], [**GM-1549**], [**GM-2014**], [**GM-2015**] **Request: Darden** **(**[**GM-1546**]** / **[**DE-975**]** / **[**133533**]**)**

- Sends Confirmation Number for In-Store Call Aheads **Jira#: **[**GM-1761**], [**GM-2132**] **Request: BJ’s** **(**[**GM-1757**]** / **[**DE-973**]** / **[**133413**]**)**

- Implement new Place In Line logic for visit confirmations **Jira#: **[**GM-2131**], [**GM-2165**] **Request: Darden** **(**[**GM-1842**]** / **[**DE-1119**]** / **[**136547**]**)**

- Updates installation to use Visual Studio 2017 **Jira#: **[**GM-1807**]

- Adds "Estimated Seat Time" to sorting method drop down in Standard Visit cells **Jira#: **[**GM-2071**]

- Supports new setting to control Takeout tab **Jira#: **[**GM-1939**], [**GM-1940**], [**GM-2119**] **(**[**GM-1937**]**)**

- Refactors DineTime Server's Polaris Sync Functionality **Jira#: **[**GM-2010**], [**GM-1289**], [**GM-1294**] , [**GM-1645**], [**GM-1646**], [**GM-1654**] **(**[**GM-1605**]**)**

- Removes the Phone Number from being displayed on the reservation visit cells **Jira#: **[**GM-2087**] **(**[**GM-1836**]**)**

- Displays "Release Order" button on Seated visits if the order has not yet been released in Kitchen **Jira#: **[**GM-2080**] **(**[**GM-1704**]**)**
