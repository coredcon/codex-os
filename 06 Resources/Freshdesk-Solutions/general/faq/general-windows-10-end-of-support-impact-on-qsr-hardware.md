# |General|  - Windows 10 End of Support: Impact on QSR Hardware

### **Overview**


Microsoft will officially end support for general versions of **Windows 10** in **October 2025**. This announcement has raised questions about operating system updates across our hardware.

### 


### **Impact on QSR Devices**


There is **no immediate impact** to the hardware currently sold by QSR. These devices include the HP Thin Client, HP Engage Flex Mini and the Elo I-series 2.0 for Windows. 


- EOL devices such as the eXpert DX-3000, Onyx, and the QSR eXceed are not supported. 


All Windows devices currently sold by QSR systems use the **Windows 10 Long Term Servicing Channel (LTSC)**. Unlike the general release of Windows 10, LTSC versions are designed for long-term stability and receive **extended support. **


### **What This Means**


- 

✅ The below **QSR hardware is not affected** by the 2025 Windows 10 end-of-life.


- 

 **No upgrade to Windows 11 is planned** at this time.


- 

 Devices will continue receiving **security updates and support** under the LTSC program.

### **Customer FAQ:**


**Q: Will I need to upgrade any QSR controllers to Windows 11?**
**A:** No. 


**Q: Can I upgrade any QSR controllers to Windows 11?**
**A:** No. There is no upgrade path available for QSR Windows controllers. 


**Q: What is LTSC?**
**A:** LTSC (Long Term Servicing Channel) is an enterprise version of Windows designed for systems that require long-term support and minimal feature changes. It is ideal for stable environments like QSR hardware deployments.


**Q:  Will my QSR Software still work if my server is on Windows 11?**
**A:** Yes. The current releases of QSR software (ConnectSmart Kitchen, ConnectSmart Host, DisplayClient, DeviceAgent, and Control Point) are certified to run on Windows 11. Please refer to the individual application's installation guide for more information.
