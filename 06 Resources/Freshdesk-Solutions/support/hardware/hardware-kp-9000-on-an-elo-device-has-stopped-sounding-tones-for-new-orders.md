# *|Hardware| - KP-9000 on an ELO device has stopped sounding tones for new orders

If you get a call from a site with an ELO backpack or ELO all in one that was purchased from QSR and is no longer sounding a tone for new orders when it was working previously please ask the following questions:


1) Did they recently update their Kitchen Server version corresponding to the issue starting?


2) Did they recently upgrade their ControlPoint version corresponding to the issue starting?


3) Is the Auto Upgrade DisplayClient version flag checked in the dataset?
4) Get the serial number and send it to Nick Jones for verification.


Depending on when the Backpack was sold it will likely need to be set up for an RMA.
