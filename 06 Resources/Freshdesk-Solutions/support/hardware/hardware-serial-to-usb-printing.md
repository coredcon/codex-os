# *|Hardware| - Serial to USB Printing

## Connecting a Serial Printer Using a USB Adapter


We offer a **USB to Serial Adapter** that allows you to connect a serial printer using a standard **Null Modem cable**.

### Supported Adapter


We currently sell the **Tera Grand USB 2.0 to RS232 Serial Adapter**. This adapter can be used to connect to the serial port of your printer via a Null Modem cable.


** **


### Driver Installation


**Note for HP Thin Clients:**
Some HP Thin Clients may require drivers to properly recognize the USB to Serial adapter. These drivers are included as attachments to this article. Please install them before attempting to connect the printer.

### Important Compatibility Note


The **ELO Backpack** does **not support** serial printers under any circumstances. This includes attempts to connect via a USB to Serial adapter.


- 

**Supported printing methods on ELO Backpack:**


- 

Direct USB printing


- 

Ethernet printing


If you're using an ELO Backpack, please ensure your printer supports one of these connection methods.
