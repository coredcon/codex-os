# *|Hardware| - OneDrive Error Message Popup on HP MP9

We have started receiving calls from sites using MP9 devices with pop ups for OneDrive that are taking focus from DisplayClient.


This is something that has always been in place however something changed and the stand alone updater is now causing an error.


If you see this message the only way to prevent it from reoccurring is to uninstall =OneDrive through the add/remove programs screen. The program is not needed for any of our software.
