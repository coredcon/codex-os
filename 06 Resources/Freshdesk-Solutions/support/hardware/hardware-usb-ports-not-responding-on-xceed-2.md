# *|Hardware| - USB Ports Not Responding On Xceed 2

Try this if you get an Xceed 2 where the USB ports are not responding


Frequently it will be an issue like the keypad lighting up and beep but not get input to the screen.


1) VNC into the device


2) Go to Start -> Run and run the xceed2mgr.


3) Disable the USB ports


4) Reboot the device


5) Reenable the USB ports


6) Reboot the device again
