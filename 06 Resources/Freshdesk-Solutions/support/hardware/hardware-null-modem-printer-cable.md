# *|Hardware| - NULL MODEM PRINTER CABLE

NULL MODEM PRINTER CABLE


The difference is the standard Serial cable is a "straight through" cable,  the pins are connected to the same pins on both ends. 


A NULL MODEM cable is a "cross wired" / "crossover" cable designed for data transfer from one device to another via a serial connection.


Our system requires the NULL MODEM cable to be in place to work. Most NULL MODEM cables have the label stamped on the cable as seen in the picture below, but this is not 100% the rule. There is not a way to tell if the cable is actually a NULL MODEM cable with out cutting into the cable to see how the wires are pinned or from the manufacture specs and packaging. Can you acquire a known good NULL MODEM cable and test with the pagers and let me know the outcome. Putty confirms the connection but not the correct data transfer.
