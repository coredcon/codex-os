# *|Hardware| - TES Touch Screen Compatibility with QSR Hardware

## Helpful Information:


It has been confirmed with JGoodhue that TES Touch Screen has not been certified to work with our hardware. There has been a reseller that attempted to use those touch screens with an xCeed and even with the manufacture CE driver it did not work.
