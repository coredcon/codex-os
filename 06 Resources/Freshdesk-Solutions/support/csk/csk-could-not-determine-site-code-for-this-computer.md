# *|CSK| - Could not determine site code for this computer

Issue:


When a customer calls in advising they receive the following error when opening the QSR License Manager:


"Could not determine site code for this computer. Please contact QSR support"


Solution:


1. Get connected to the customer's primary kitchen server PC.


2. Ask and confirm with the customer that they do not have virtual network drives attached and or that IPv4 isn't disabled.


3. Uninstall ConnectSmart Kitchen server from the primary kitchen server and re-install with admin credentials.
