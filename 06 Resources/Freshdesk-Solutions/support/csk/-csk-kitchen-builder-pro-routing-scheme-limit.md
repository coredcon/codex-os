# * |CSK| - Kitchen Builder Pro Routing Scheme Limit

## **Issue:**

## Additional Routing Schemes cannot be created. 

## 


## **Solution:  **


There is a limit to how many routing schemes you can have created in Kitchen Builder Pro.  Under "Routing Schemes" the max number of schemes is 128. If there is a need for  existing schemes to create new ones.
