# *|CSK| - Creating Menu Cards To Display Item Images 

This article will help set up a site to view an Item's image using the "Item Image" option on their Keypad or Dashboard Templates. 


Assign Item Image button to Keypad or Dashboard. 


File path will be, C:\ProgramData\QSR Automations\ConnectSmart\KitchenServer\Images\MenuCards


Create the "MenuCards" subfolder in Images if it does not exist. 


Save the item images in the MenuCards subfolder as "Item**XXXXX**.png" 


*****XXXXX = Item ID (PLU).*****


This is case sensitive, do not forget the upper case in** ****M**enu**C**ards and **I**temXXXXX.png
