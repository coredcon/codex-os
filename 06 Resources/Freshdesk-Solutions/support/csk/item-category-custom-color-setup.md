# Item Category - Custom Color setup

When a customer calls in requesting items to show up on the kitchen station using specific colors, use the steps below. 


**Step 1:**
Once connected to the customer's server, open ConnectSmart Kitchen Builder Pro and choose the desired dataset to review.


**Step 2:**
Confirm that Item Categories has the item setup with the Category value and desired color.


**Step 3:**
Confirm that the "Item Attributes Priority Templates" Has the "Item Category" moved to the very top of the list.

**Step 4: **
Confirm that the desired station the customer is working on is using the correct "Item Attributes" Template.


**Step 5:**


Confirm the POS software is sending the desired Item with the configured Item Category Value setup in ConnectSmart Kitchen Builder pro.
