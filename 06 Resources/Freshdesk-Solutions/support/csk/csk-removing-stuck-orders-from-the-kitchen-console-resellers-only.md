# |CSK| - Removing Stuck Orders from The Kitchen Console - Resellers Only

This article will assist you in removing stuck orders from the screen by deleting the transaction through the Kitchen Console. This is for Internal and Reseller use ONLY and should not be provided at a site level.


- Open the** ConnectSmart Network Configuration** tool and verify the **Primary Kitchen Server** IP address.


- Make sure that Kitchen Server is running (It does not matter if it is running as a process or as a service, it just has to be running).


-  Navigate to the following URL (Replace the XXX.XXX.XXX with the primary Server IP):  **http://XXX.XXX.XXX:32768/KitchenConsole/KitchenConsole.html**


- Once ConnectSmart Kitchen Console has loaded, select the **System **tab.


- Select the Transactions tab


- Locate the transaction needing to be removed and click Remove
**IMPORTANT NOTE:** You will need to remove each listing for the transaction if there are multiple entries.
