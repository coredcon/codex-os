# *|CSK| - Kitchen Builder Pro Display Issue Fix for Smaller Screens

**Issue:**


Some users have reported an issue within Kitchen Builder Pro with windows not scaling properly when a smaller monitor is in use. This issue caused many settings to not be visible to the user. 


**Resolution: **


1.  Navigate to C:\Program Files (x86)\QSR Automations\ConnectSmart\KitchenServer\Bin


2.  Right click the KitchenbuilderPro.exe and select “Properties”


3.  Select the “Compatibility” tab


4.  Select “Change high DPI settings”


5.  Tick “Override high DPI scaling behavior”


6.  Select “system” from the drop down box.
