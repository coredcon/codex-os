# *|Connect Applications| - Installing POSitouchTsConnect CSK

Installation Notes:


In order to properly set up POSitouchTSConnect will need to create the following directories based for CKS: 


 


 Configuration:


1) Create the following directories:


        C:\Program Files (x86)\QSR Automations\POSitouchTsConnect\Bin


        C:\Program Files (x86)\QSR Automations\POSitouchTSConnect\Data


        C:\Program Files (x86)\QSR Automations\POSitouchTSConnect\Courses


        C:\Program Files (x86)\QSR Automations\POSitouchTSConnect\Log 


2) Drop"POSitouchTsConnect.exe" and "POSitouchTSConnectEditor.exe" into the Bin folder you created in Step 1.


3) Run the"POSitouchTSConnectEditor.exe" and set up as needed per the directions in the user guide.
