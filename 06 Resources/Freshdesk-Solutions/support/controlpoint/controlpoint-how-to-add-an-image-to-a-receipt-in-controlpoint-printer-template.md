# *|ControlPoint| - How to add an image to a receipt in ControlPoint printer template

## Goal/Desired Behavior:


With ControlPoint 4 and newer clients can add images to their receipt.


## Steps:


- Drop the image file into the following path:

- C:\ProgramData\QSR Automations\ConnectSmart\ControlPointServer\ReceiptImages


- Open Control Point > Click on printer setting icon > Highlight printer you want to add the image to > Edit > Select icon > Select location > Check “Make Selected Image Default” > Apply
