# *|ControlPoint| - System Requirements for Controlpoint 

## Controlpoint 3.3+ Min requirements


**ControlPoint Server:**


.NET 4.6.1, VCRuntime 2005, VCRuntime 2015


**ControlPoint Client:**


.NET 4.6.1 (no C++ prereq)


**OS:**


Win7 SP1
