# Troubleshooting ELO AIO Not Showing Online In ControlPoint

For ELO AIO Devices, since recent ControlPoint release they should be showing as online in ControlPoint.


**Troubleshooting**


- Make sure on Correct version of ControlPoint

- Confirm Windows Firewall on server disabled

- On DispalyCleint switch device to TCP connection

- If that does not work Switch Back to UDP connection

- Try restarting Device to make sure that any manual settings have been cleared out.
