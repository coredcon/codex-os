# *|CSH| - Unable to See the Active Floor Plan Image

**Issue**: 


The customer states that they are unable to seat guests or see the Floor Plan image in the ConnectSmart Host client. 


**Solution**: 


- Make sure a shift has been selected from the Servers tab in the Windows client or the Sections tab from the iOS app.

- If there are no shifts available, please ensure that a shift has been built either in the Host portal or via the Host client.  

- If there has been a recent floor plan change, then the shift will need to be reset. 


If the customer would like a certain shift to be active after EOD, they can configure this in the Host Portal. 


- Navigate to the Host Portal > Sites > Customize > Site Settings > Change Shift at End of Day

- Flag the days that you want to enable this feature and select the desired shift from the drop-down.
