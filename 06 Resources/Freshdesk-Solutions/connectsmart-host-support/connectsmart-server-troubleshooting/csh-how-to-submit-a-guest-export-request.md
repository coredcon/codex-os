# |CSH| - How to Submit a Guest Export request

**Issue:**Customer needs to get a full export of their Guestbook. Please direct the customer to the following path for completion. 


**Solution: **The process for requesting an export of the Guestbook is now done through the Company Portal.   go the following path: [Company Portal] > Marketing > Guest Export. The user will then be presented with a screen to submit the request and enter which email you would like to receive the link.[]


**** Please note that the user is informed of a 3 business day turn around for the request to be completed. ****


Once the user submits the request, the Technical Projects team automatically receives a ticket in their queue. They will then create a resource request for the BIR team to complete the export. The Technical Projects team will reply to the customer with the provided CSV file.
