# *|CSH| - How to tell if ConnectSmart Host customer is "Server" or "Cloud"

How to tell if DineTime customer is "Server" or "Cloud"


- DineTime Server customers will have a license in LMTS

- The host app will ask specifically for either the Email address or the Username on Login

- Server customers auto-login (if set in the Terminal Roles) or use Username and Password

- Cloud Customers log in with Email Address and Password


- You can verify the type in the Vue Admin. 

    1) Login to the Vue admin at [https://admin.qsr.cloud/]


    2) Navigate to the desired site from the 'Sites' tab 


    3) Expand Site Settings > General > DineTime Type will show DineTime server or DineTime Cloud
