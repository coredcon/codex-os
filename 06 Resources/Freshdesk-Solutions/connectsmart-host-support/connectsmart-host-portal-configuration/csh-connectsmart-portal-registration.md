# *|CSH| - ConnectSmart Portal Registration

## How to Register a Site with the Portal


1.  Login at [https://portal.qsr.cloud/]


2. On the Sites tab, click on the site needing registered:


3. On the Manage tab, choose Server Registration from the left-hand menu


4. If there is no registration code already, click on "Generate Dinetime Code"


5. On the server, open the ConnectSmart Site Registration tool.


6. Enter the Registration code that was generated in the Portal in step 4 and click **Register**.
