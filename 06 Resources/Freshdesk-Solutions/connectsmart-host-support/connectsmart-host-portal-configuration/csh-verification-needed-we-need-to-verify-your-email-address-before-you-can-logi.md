# *|CSH| -  "Verification Needed - We need to verify your email address before you can login" on Portal Login

**User receives the following message when logging into the Portal:**


 


Verification Needed


We need to verify your email address before you can login.


When you create a new account or change your email, we automatically send you an email requesting verification. To verify, just click the link in the email. When you're done, go back and try signing in again.


**Solution:**


- When a new Portal User is created an email with a verification link is sent to the email address.  The User must click this link to activate the User account.

- If the User believes they have not received the verification link, below the "Verification Needed" message there is a link labeled "Send verification email again".

- If the User still does not receive the email, have them check their spam/junk folder.

- We have the ability to activate the User on the backend, but every attempt should be made to have the User actually verify the email address before using this option.
